# 🌐 Pawan Mishra – Developer Portfolio

This is my personal **developer portfolio website** built using modern web technologies.
It showcases my **projects, skills, and experience** as a developer.

## 🚀 Tech Stack

* **React**
* **Vite**
* **TypeScript**
* **Tailwind CSS**

## 📂 Project Structure

```
Portfolio_mine
│
├── public          # Static assets
├── src
│   ├── components  # UI components
│   ├── pages       # Main pages
│   ├── hooks       # Custom hooks
│   ├── lib         # Utility functions
│   ├── App.tsx
│   └── main.tsx
│
├── index.html
├── package.json
├── tailwind.config.ts
└── vite.config.ts
```

## 🛠️ Run Locally

Clone the repository:

```
git clone https://github.com/mishraj018/Portfolio_mine.git
```

Go to the project folder:

```
cd Portfolio_mine
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

The website will run on:

```
http://localhost:8080
```
